Preguntas Frecuentes

¿Qué pasa si la IA comete un error?

Healthpixel opera como asistente, no como decisor. Todas las reservaciones requieren datos completos validados por esquemas Zod. El sistema no puede crear reservaciones con datos incompletos o inválidos.

¿Es seguro usar WhatsApp para datos médicos?

Los datos sensibles del paciente (RUT, teléfono) se encriptan con AES-256 antes de almacenarse. WhatsApp se usa como canal de comunicación; los datos clínicos se procesan y almacenan en el servidor dedicado.

¿Qué sucede si el sistema se cae?

El sistema incluye monitoreo automático, reconexión de WhatsApp con retry exponencial, y almacenamiento atómico con file locking que previene corrupción de datos. SLA de disponibilidad incluido en contrato.

¿Se puede integrar con nuestro sistema actual?

El sistema opera de forma independiente y no requiere integración con ERP existente. Los doctores interactúan vía WhatsApp y los coordinadores vía el dashboard web.

¿Cómo se capacita al personal?

La capacitación es mínima: los doctores hablan con el agente por WhatsApp como lo harían con una persona. Para coordinadores, incluimos sesión de 2 horas sobre el dashboard.
